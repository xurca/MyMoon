## Enable Migrations

1. Install-Package Microsoft.EntityFrameworkCore.Tools
2. Add-Migration Update-Database Script-Migration
3. [migrations tutorial](https://www.learnentityframeworkcore.com/migrations/commands/pmc-commands) 

## Installation

1. [PostgreSQL](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads)
2. dotnet-hosting-3.1.3-win
3. dotnet-sdk-3.1.201-win-x64
4. to install database run command **Update-Database** from Package Manager Console
